学员评价

张华/四川大学锦江学院

学车的过程是温暖有趣的，与两位罗教练从初识到结识从而成为朋友，对我来说，非常的幸运并且心怀感恩，无论是学车技巧上，还是生活中而言，亦是良师也是益友。在学车过程中，大家相处的氛围非常的轻松融洽，很开心融入这个大家庭，学车生涯告一段落啦，但我们友谊不散。

刘钟林/四川托普信息技术职业学院

我们场地有三个叮叮的师傅，其中一个李教是我觉得最好的教练，一个幽默的教练，对学员负责，而且也对学员很好，可以说是一个幽默的暖男吧，其他还有张教待我的次数不多，但也是很耐心指导的，另一个罗教很严格，一点失误都要挑出来，性格还是有点严肃的，但总体感觉还可以。

张丽/四川大学锦城学院

我是我们学车班拿证最快的，一个多月就搞定驾照，主要都是教练教的很好啦，同班的学友都觉得邹教练是个耐心的暖男，记得有个同学把教练车给碰坏了，他一点不生气反而还安慰学员，真的是为叮叮的教练打call~

冯春蓉/四川大学锦江学院

接送很及时，我练车的时候是夏天，教练每天早上都会在后校门准时接我们，练完车又送我们回来，场地上偶尔会看到有其他驾校的师傅在骂人，但我们师傅从来不骂人，而且还会给你讲很多他的经历，教你一些生活哲学。
